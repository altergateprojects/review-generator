## 🆘 Troubleshooting

### Reviews not generating:
1. Check environment variables in Vercel
2. Check Gemini API key is valid
3. Check API quota hasn't been exceeded
4. Check Vercel function logs

### QR codes not working:
1. Verify URL is correct
2. Test URL in browser first
3. Regenerate QR codes if needed
4. Clear phone camera cache

### Wrong business showing:
1. Check URL includes business ID
2. Verify config.js has correct business IDs
3. Clear browser cache
4. Redeploy if needed

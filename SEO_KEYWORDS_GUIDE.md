# 🎯 SEO Keywords Guide for Better Google Ranking

## Why Keywords Matter

Including relevant keywords in Google reviews helps your business:
- **Rank higher** in local search results
- **Appear in more searches** (e.g., "best resort near me")
- **Match customer search intent**
- **Improve visibility** on Google Maps

## ✅ How It Works

The AI will **naturally incorporate** 1-2 keywords from your list into each generated review, making them sound authentic and helpful (not spammy).

## 🔧 How to Customize

Edit `config.js` and update the `SEO_KEYWORDS` array:

```javascript
SEO_KEYWORDS: [
  "Royal Shourya Resort",           // Always include your business name
  "best resort in [your city]",     // Location-specific
  "luxury family resort",           // Your unique selling point
  "weekend getaway destination",    // Use case
  "5-star resort",                  // Quality
  "resort with spa and pool"        // Key amenities
]
```

## 💡 Best Practices

### ✅ DO:
- Include your **business name** (most important!)
- Use **natural phrases** people actually search for
- Include **location** if relevant
- Mention **key amenities** or features
- Use **3-6 keywords** (don't overdo it)
- Think about what **customers search for**

### ❌ DON'T:
- Stuff too many keywords (looks spammy)
- Use unnatural phrases
- Include competitor names
- Use misleading terms
- Repeat the same keyword multiple times
